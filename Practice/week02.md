# 목표

AI를 활용하여 디자인을 변경해보자

# Q1: 현제코드에서 디자인을 예쁘게 변경해줘!

- AI답변: 총 2가지의 디자인요소를 추가했고 각각을 star와 Particle클래스로 만들어서 쉽게 관리하고독 변경함, star는 클릭하지 않아도 위쪽에서 랜덤한 위치에 아래로 떨어지는 하얀 원이고 particle로는 클릭하였을때 블랙홀,중력,노이즈 등의 효과를 추가해 화려한 디자인을 위한 코드를 작성해줌.

- 적용: 설명대로 디자인이 잘 나옴.

# Q2: 깃 푸쉬할때 래파스토리지가 내 권한이 아니라고 푸쉬하지 못함.

- AI답변: git config 정보를 수정하도록 안내해줌.

- 시행착오: 이미 git config는 학교 계정으로 설정된 상태

- 재질문: 내가 이미 쓰던 계정이 있고 학교 계정으로 깃헙에 과제를 올려야 하는데 안됨.

- AI답변: 커밋계정과 푸쉬계정은 별게이기 때문에 푸쉬계정을 지우고 다시 로그인 하거나, ssh 키를 설정하도록 함

- 최종해결: 2개의 계정에 ssh-key를 사용함으로서 url 대신 .ssh에 만들어둔 config 별칭으로 쉽게 다양한 계정을 사용할수 있음

# 배운것

- 깃은 기본적으로 하나의 계정만 사용가능하다
- 동시에 여러계정을 다루고 싶다면 ssh키를 사용할 모든 계정에 설정하여 쉽게 변경할수 있다.
